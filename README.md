# Tyl
Tyl is a web-based 2D tile map editor designed for platformer games.
The goal of this project is to explore game development tooling using
modern web technologies.

## Current Features
- Canvas-based grid rendering
- Modular project architecture
- JSON-based map format (WIP)

## Tech Stack
- TypeScript
- React
- Vite
- HTML5 Canvas
- Tailwind CSS

## Roadmap
- Tile placement and layers
- Tileset loading
- Map export (JSON / PNG)
- Editor tools (undo, presets)
